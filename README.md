# 🎮 Rock Paper Scissors - Python

This is a simple **Rock, Paper, Scissors** game developed in Python as part of my programming learning journey.

The goal of this project is to practice **programming logic, control structures, and user input validation** while also improving my **English skills**, since the entire code and interaction with the user are written in English.

##  Features

* User input validation
* Random choice for the opponent
* Game loop allowing multiple rounds
* Win / Lose / Tie logic
* Exit option to end the game

##  What I practiced

* Loops (`while`)
* Conditional statements (`if`, `elif`, `else`)
* Input validation
* Python modules like `random` and `re`
* Basic game logic
* Writing code and messages in English

* Python
